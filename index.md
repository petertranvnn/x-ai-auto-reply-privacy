Data Collection & Usage
X AI AUTO REPLY does not collect, store, or share any personal or sensitive data from users.
The extension only processes the following locally on your device:
- Text content from the X (Twitter) page currently being viewed
- Your local settings such as style, mode, prompt suggestions (saved in chrome.storage.local)

External API
When you click the AI button, the extension sends the text context (tweet/reply text and your settings) to OpenAI API to generate a response.
No personal identifiers are sent intentionally.
All communication uses encrypted HTTPS.

No Tracking, No Selling
- No analytics
- No advertising
- No profiling
- No selling or sharing data with third parties

User Control
You can remove or reset extension data anytime via:
- Extension options (Reset settings)
- Removing the extension from Chrome

Permissions Use Explanation
- scripting: Required to insert AI suggestions into the reply box
- activeTab: Only to interact with the currently open X tab when user presses AI button
- storage: Save user settings locally
- host_permissions: Only for https://x.com/* and https://api.openai.com/*
